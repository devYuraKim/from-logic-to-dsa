# Dynamic Programming

**이미 계산한 거 어딘가에 적어두고, 나중에 또 필요하면 다시 계산 안 하고 꺼내 쓰는 것**

### 그래서 이거 언제 쓰냐?

대표적 예시: **피보나치 수열**

```text
fib(5) 를 구하려면?

fib(5) = fib(4) + fib(3)
fib(4) = fib(3) + fib(2)
fib(3) = fib(2) + fib(1)
...
```

$n=5$일 때, `fib(3)`은 두 번, `fib(2)`은 세 번 계산. 트리가 커질수록 이 중복이 폭발.

"**이미 계산한 거 어딘가에 적어두면 안 돼?**" > **DP**

내가 알고 있는 건 React의 `useMemo`: 같은 입력값이면 다시 계산 안 하고 캐시된 값 반환하는 것.

끝.
